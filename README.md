# Calculator Project

## Overview

A basic on-screen calculator built with **HTML**, **CSS**, and **JavaScript**, completing the final project of the Foundations course from [The Odin Project](https://www.theodinproject.com/lessons/foundations-calculator/).

## Features

- **Basic arithmetic**: addition, subtraction, multiplication, and division
- **Chaining of operations**: e.g., `12 + 7 - 1 =` computes sequentially
- **Decimal support**: input of floating point numbers and prevention of multiple decimals
- **Clear functionality**: resets all values and display
- **Error handling**: e.g., snarky message when dividing by zero
- **Display overflow handling**: rounding or formatting long numbers to fit the screen
- **Backspace feature**: delete the last entered digit
- **Keyboard support**: use keys for digits, operators, clear, equals, etc.

## Built With

- HTML
- CSS
- JavaScript
